## 🔧 部署流程优化建议

### 当前问题
- ❌ 手动同步容易遗漏
- ❌ 无版本控制
- ❌ 无自动化测试

### 建议方案

#### 方案 1: 自动挂载（推荐）
```yaml
# docker-compose.yml
volumes:
  - ./mobile:/usr/share/nginx/html/qc-mobile:ro
```

**优点**:
- ✅ 实时同步
- ✅ 无需手动操作
- ✅ 开发友好

#### 方案 2: 部署脚本
```powershell
# deploy.ps1
$files = Get-ChildItem "./mobile/*.html"
foreach($f in $files) {
  docker cp $f.FullName "qc-nginx:/usr/share/nginx/html/qc-mobile/$($f.Name)"
}
docker exec qc-nginx nginx -s reload
```

#### 方案 3: CI/CD
- Git push 触发
- 自动构建镜像
- 自动部署到服务器
