                "solution_type": i.solution_type,
                "compensation_amount": float(i.compensation_amount) if i.compensation_amount else 0,

## 🔧 5. 修复方案

### 5.1 修复 API 接口

**修改 `/app/backend/main.py`**:
```python
@app.get("/api/issues", response_model=Dict[str, Any])
async def get_issues(...):
    # ...
    return {
        "total": total,
        "page": page,
        "page_size": page_size,
        "data": [
            {
                "id": i.id,
                "issue_no": i.issue_no,
                "goods_no": i.goods_no,
                "factory_name": i.factory_name,
                "issue_type": i.issue_type,
                "issue_desc": i.issue_desc,
                "solution_type": i.solution_type,
                "compensation_amount": float(i.compensation_amount) if i.compensation_amount else 0,
                "product_image": i.product_image,  # ✅ 添加
                "issue_images": i.issue_images,    # ✅ 添加
                "created_at": i.created_at.isoformat() if i.created_at else None
            }
            for i in issues
        ]
    }
```

### 5.2 修复 created_at 为 NULL 的问题

**问题**: 数据库 created_at 字段为 NULL

**原因**: 测试数据插入时未设置 created_at

**修复**:
```sql
-- 更新测试数据的 created_at
UPDATE quality_issues 
SET created_at = NOW() 
WHERE created_at IS NULL 
  AND issue_desc LIKE '%图片测试%';
```
