## 🔗 4. API 数据源验证

### 后端代码检查

**API 接口**: `/app/backend/main.py`

```python
@app.get("/api/issues", response_model=Dict[str, Any])
async def get_issues(
    page: int = 1,
    page_size: int = 20,
    factory: Optional[str] = None,
    issue_type: Optional[str] = None,
    current_user: QCUser = Depends(get_current_user),
    db: Session = Depends(get_db)  # ← 使用数据库会话
):
    query = db.query(QualityIssue)  # ← 从数据库查询
    
    if factory:
        query = query.filter(QualityIssue.factory_name == factory)
    if issue_type:
        query = query.filter(QualityIssue.issue_type == issue_type)
    
    total = query.count()  # ← 数据库计数
    issues = query.order_by(QualityIssue.created_at.desc())\
                  .offset((page - 1) * page_size)\
                  .limit(page_size)\
                  .all()  # ← 数据库查询
    
    return {
        "total": total,
        "page": page,
        "page_size": page_size,
        "data": [
            {
                "id": i.id,
                "issue_no": i.issue_no,
                "goods_no": i.goods_no,
                "factory_name": i.factory_name,
                "issue_type": i.issue_type,
                "product_image": i.product_image,  # ← 数据库字段
                "issue_images": i.issue_images,    # ← 数据库字段
                "created_at": i.created_at.isoformat() if i.created_at else None
            }
            for i in issues
        ]
    }
```

**验证**:
- ✅ 使用 `db: Session = Depends(get_db)` 数据库会话
- ✅ 使用 `db.query(QualityIssue)` 查询数据库
- ✅ 使用 `query.count()` 数据库计数
- ✅ 使用 `query.all()` 数据库查询
- ✅ 返回数据来自数据库模型字段

**结论**: ✅ **API 直接连接数据库，无缓存，无硬编码**
