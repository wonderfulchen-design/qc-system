### 2.2 重点问题类型详解

#### 🔴 污渍问题 (26.7%, 预估 26,584 条)

**问题表现**:
- 商品表面有明显污点、油渍、水渍
- 多发生在袖口、下摆、前襟等部位
- 部分污渍无法清洗去除

**根本原因**:
1. 生产环境灰尘多，防尘措施不足
2. 工人操作不规范，手污/工具污
3. 包装前未进行最终检验
4. 仓储环境潮湿，导致霉斑

**影响**:
- 客户满意度严重下降
- 退货率高达 80%
- 平均补偿 RMB 10-20/件

#### 🔴 做工开线问题 (20.0%, 预估 19,913 条)

**问题表现**:
- 缝线开裂，多发生在接缝处
- 线头未处理，影响美观
- 针距不均匀，牢度不足

**根本原因**:
1. 缝制工艺参数不当 (针距/线张力)
2. 工人技能不足，培训不到位
3. 检验标准执行不严格
4. 为赶工期忽视质量

**影响**:
- 影响穿着，安全隐患
- 退货率约 60%
- 品牌声誉受损

#### 🟡 起球勾线掉毛 (13.3%, 预估 13,242 条)

**问题表现**:
- 面料表面起球
- 针织物勾丝
- 毛绒类掉毛严重

**根本原因**:
1. 面料质量不达标
2. 后整理工艺不当
3. 未进行起球测试
4. 包装方式不当

### 6.3 检验流程设计

#### 6.3.1 来料检验 (IQC)

**检验对象**: 面料、里料、辅料 (扣子/拉链/洗唛等)

**检验标准**:
- 面料：色差≤4 级、瑕疵≤3 处/100 米、克重偏差±3%
- 辅料：尺寸公差±0.5mm、色牢度≥4 级

**检验流程**:
```
供应商送货 → 核对送货单 → 抽样 (AQL 2.5) → 检验 → 
  ├→ 合格：入库
  └→ 不合格：退货/特采 → 记录供应商档案
```

**关键控制点**:
- 每批次面料必须检验
- 新供应商首批全检
- 建立供应商质量档案

#### 6.3.2 制程检验 (IPQC)

**检验对象**: 裁剪片、半成品、成品

**检验标准**:
- 裁剪：裁片尺寸公差±0.5cm、无瑕疵
- 车缝：针距 3cm/12-14 针、无跳针/浮线
- 整烫：无烫痕、无极光、尺寸符合

**巡检频率**:
- 裁剪：每款首件必检，过程中抽检 10%
- 车缝：每工序首件必检，过程中巡检 2 小时/次
- 后整：每件必检

**异常处理**:
```
发现异常 → 立即停机 → 通知班组长 → 
  ├→ 小问题：现场纠正
  └→ 大问题：隔离不合格品 → 质量工程师分析 → 纠正预防措施
```

#### 6.3.3 最终检验 (FQC)

**检验对象**: 成品服装

**检验标准**: AQL 2.5 (致命缺陷 0，严重缺陷 2.5，轻微缺陷 4.0)

**检验项目**:
1. 外观检验 (污渍/破损/色差/线头)
2. 尺寸检验 (关键部位尺寸)
3. 工艺检验 (缝制/整烫/包装)
4. 标识检验 (洗唛/吊牌/条码)

**检验流程**:
```
成品入库 → FQC 抽样 → 检验 → 
  ├→ 合格：贴合格证，入成品库
  └→ 不合格：返工/降级 → 重新检验
```

#### 6.3.4 出货检验 (OQC)

**检验对象**: 即将出货的成品

**检验标准**: AQL 2.5

**检验流程**:
```
收到出货通知 → 按批次抽样 → 检验 → 
  ├→ 合格：放行出货
  └→ 不合格：整批返工 → 重新检验
```

### 6.4 质量考核体系

#### 6.4.1 工厂考核指标

| 指标 | 权重 | 目标值 | 计算方法 | 数据来源 |
|------|------|--------|----------|----------|
