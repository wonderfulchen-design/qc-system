### 1.2 单选逻辑实现

**JavaScript 代码**:
```javascript
let selectedSolution = null;

function selectSolution(element) {
    // 1. 清除所有选项的选中状态
    document.querySelectorAll('.solution-option').forEach(opt => 
        opt.classList.remove('selected')
    );
    
    // 2. 选中当前点击的选项
    element.classList.add('selected');
    
    // 3. 保存选中的值
    selectedSolution = element.dataset.value;
}
```

**单选机制**:
1. ✅ 点击时先清除所有选项的选中状态
2. ✅ 只选中当前点击的选项
3. ✅ 只保存一个值到 `selectedSolution`
4. ✅ 提交时只传递单个值

**验证结论**: ✅ **实现正确，是单选！**
