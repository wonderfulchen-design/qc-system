## 🎯 两种测试方案

### 方案 1: Python Selenium 自动化（推荐）

**文件**: `auto_submit_test.py`

**优点**:
- ✅ 完全自动化
- ✅ 模拟真实用户操作
- ✅ 可视化操作过程
- ✅ 可录制截图/视频

**安装依赖**:
```bash
pip install selenium
pip install webdriver-manager
```

**使用方法**:
```bash
# 提交 10 条
python auto_submit_test.py

# 提交 20 条
python auto_submit_test.py 20
```

**功能**:
1. 自动打开浏览器
2. 自动登录（admin/admin123）
3. 跳转到问题录入页
4. 批量提交 10 条问题
5. 显示进度和统计
6. 提交完成后保持浏览器打开

### 方案 2: 批量提交工具页面

**文件**: `batch-submit-tool.html`

**优点**:
- ✅ 无需安装
- ✅ 直接用浏览器打开
- ✅ 图形界面友好
- ✅ 实时日志显示

**使用方法**:
```
1. 用浏览器打开 batch-submit-tool.html
```
